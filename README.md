## Ikujitime / Memo-me

Web app helping the user to keep tabs on the kids around him

## After cloning

```bash
rails db:create db:migrate db:seed
bundle install
yarn install
```

## Usage

set up your own Cloudinary-key and Mapbox-api-key inside your ".env" file

```python
CLOUDINARY_URL=cloudinary://***********************************************************
MAPBOX_API_KEY=********************************************************
```
